# Employee Management System (Python)

This is a simple Employee Management System built using Python in a Jupyter Notebook.
The project demonstrates basic concepts such as dictionaries, functions, loops, and user input.

## Project Description
This project allows the user to manage employee data using a simple menu-based system.
Each employee is stored using an ID and includes the following information:
- Name
- Position
- Salary
- Email

## Features
- Add a new employee
- View all employees
- Update employee data
- Delete an employee
- Search for an employee by ID

## Technologies Used
- Python 3
- Jupyter Notebook

## Project Structure
project/
│── employee_system.ipynb
│── README.md

## How the Project Works
- Employees are stored in a dictionary called employees.
- The key is the employee ID.
- The value is another dictionary containing employee details.
- The system runs inside a loop until the user chooses to exit.

## How to Run the Project
1. Open Jupyter Notebook.
2. Open the file employee_management.ipynb.
3. Run the cells in order.
4. Use the menu to manage employees.
